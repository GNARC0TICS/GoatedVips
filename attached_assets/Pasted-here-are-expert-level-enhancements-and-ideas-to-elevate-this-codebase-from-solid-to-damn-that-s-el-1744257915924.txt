4. Code Enhancements

A. Custom Hook: useMVPData

Move your normalization logic inside a hook:

const useMVPData = (leaderboardData) => {
  return useMemo(() => {
    // existing normalization logic here...
  }, [leaderboardData]);
}

Cleans up the main component and makes testing easier.
